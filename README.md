# AI Resume Parser
AI Resume Parser is a web application that analyzes resumes and provides insights based on skills, certifications, GitHub, LinkedIn profiles, and resume quality.
## Features
- Resume upload (PDF)
- Skill extraction
- Certification detection
- GitHub profile detection
- LinkedIn profile detection
- Resume scoring
- Attractive dashboard interface
## Technologies Used
- Python
- Flask
- HTML
- CSS
- PDF Processing
